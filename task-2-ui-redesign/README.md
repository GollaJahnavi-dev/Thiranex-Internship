# 🎬 Task 2 - Website UI Redesign (Netflix Inspired)

## 📌 Project Overview
This project is developed as part of the **Thiranex Internship Program**.  
It focuses on redesigning a Netflix-inspired user interface using **HTML and CSS** to improve layout structure, visual hierarchy, and user experience.

---

## 🎯 Objective
The main objective of this task is to:

- Redesign an existing OTT-style interface  
- Improve visual hierarchy and content structure  
- Build a responsive layout using frontend technologies  
- Enhance user experience using clean UI design  

---

## 🛠️ Tools & Technologies Used

- HTML5  
- CSS3  
- Visual Studio Code (development environment)  
- Web Browser (for testing and execution)  

---

## 💻 Implementation Details

The project is fully implemented using **static web technologies (HTML & CSS)**.

### Features Implemented:
- Navigation bar layout  
- Hero section with featured content  
- Category-based content sections:
  - Trending Now  
  - Recommended For You  
  - Top Movies in India  
- Card-based UI structure for movie thumbnails  
- Horizontal scroll layout for content rows  
- Hover effects for interactive feel  
- Responsive design for mobile and desktop screens  

---

## 📱 Responsive Design

The interface adapts to different screen sizes:

- Desktop layout with multi-row grid structure  
- Mobile layout with stacked navigation and scrollable rows
